# Calculator-Numerical-methods-
In this project, I implemented 8 Numerical Analysis methods using Python.
The implemented methods are categorized as follows:

Root Finding Algorithms:
Bisection Method
False Position Method
Newton-Raphson Method
Secant Method
Fixed Point Iteration Method

Matrix and Linear Equation Solvers:
Gaussian Elimination
Partial Pivoting
LU Decomposition
These methods were developed and tested to solve nonlinear equations, systems of linear equations, and matrix-related computational problems efficiently.
